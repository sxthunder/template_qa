# TEMPLATE_QA

eg：

输出解析：患病时间大于20180000,and,患病时间小于20190000,and,诊断名称不等于肝脏功能异常,or,诊断名称不等于急性肾盂肾炎,and,身高等于180,and,体重大于50,and,性别等于女
